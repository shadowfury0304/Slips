Q1]
Write a shell script to take a file from the user as an argument and disk usage by that file in human readable format.
=>
# Check if a filename is provided as an argument
if [ $# -eq 0 ]; then
    echo "Usage: $0 filename"
    exit 1
fi
# Assign the first argument to a variable
file="$1"
# Check if the file exists
if [ ! -e "$file" ]; then
    echo "File '$file' does not exist."
    exit 1
fi
# Display disk usage in human-readable format
du -h "$file"


Q2]
I want to create a studentName_PRN.txt file on Desktop. Now I want to print information of files and directories present on Desktop on terminal and at the same time write that information in studentName_PRN.txt Write a shell script for this.
=>
FILE=~/Desktop/studentName_PRN.txt
# Create the file if it doesn't exist
touch "$FILE"
# Print information of Desktop files/directories on terminal and write to file
ls -l ~/Desktop | tee "$FILE"
echo "Information saved to $FILE"

running processes into it.
=>

Q6]
Write a shell script to accept process ID from the user and kil process with that ID.
=>
read -p "enter the pid to kill:" pid
expr "$pid"+0 /dev/null 2>&1
if[$? -ne 0];then
echo "error:pid must be in number";
exit 1
fi
kill -0 "$pid" 2>/dev/null
if[$? -ne 0];then
echo "pid did not exist";
exit 1
fi
kill -9 "$pid"
if[$? -eq 0];then
echo "process killed"
else
echo "process not killed"
fi


Q7]
Write shell script which will accept process nae from user and kill all processes with that 
name. 
=>
read -p "enter the process name:" pname
if[-z "$pname"];then
echo "process name cant be empty"
exit 1
fi
pids=$(grep "$pname")
if[-z "$pname"];then
echo "no process found with name:$pname"
exit 1
fi
kill -9 $pids
echo"all pids with name '&pname' have been killed"
